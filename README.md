# CodeAlpha Stock Portfolio Tracker 📊

## 📌 Internship Project
This repository contains **Task 2: Stock Portfolio Tracker** completed as part of my **Python Internship at CodeAlpha**.  
The project demonstrates Python concepts such as dictionaries, loops, conditionals, arithmetic operations, and file handling.

---

## 📝 Project Description
The Stock Portfolio Tracker allows users to:
- Enter stock symbols and quantities.
- Calculate the total value of their portfolio based on predefined stock prices.
- Display a summary of holdings.
- Save the report to a text file (`portfolio.txt`).

---

## ⚙️ Technologies Used
- Python 3
- Dictionaries
- Loops & conditionals
- File handling
